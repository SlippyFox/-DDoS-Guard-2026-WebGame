# 🛡 DDoS-Guard Ninja Pro

Веб-игра в стиле "Fruit Ninja", разработанная с использованием современных технологий для демонстрации механик защиты и взаимодействия с брендом DDoS-Guard.

## 🚀 О проекте
Это интерактивное приложение, где игроки должны "разрезать" компьютерные вирусы, защищая маскотов компании. Проект объединяет в себе игровую механику, систему рейтинга и интеграцию с Telegram.

### Основные фичи:
* **Игровой движок Phaser 3**: Плавная механика разрезания объектов и физика.
* **Интерфейс на Vue 3**: Реактивное управление состояниями (регистрация, лидерборд, админка).
* **Telegram Auth**: Регистрация пользователей через бота.
* **Backend**: Node.js + Express с базой данных SQLite для хранения очков.
* **Brand Style**: Полное соответствие гайдлайнам DDoS-Guard (цвета, шрифты, маскот).

## 🛠 Технологический стек
* **Frontend**: HTML5, CSS3 (Custom animations), Vue.js 3, Phaser 3.60.
* **Backend**: Node.js, Express.
* **Database**: SQLite 3.
* **Bot API**: Telegraf.

## 📦 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/your-username/ddos-ninja-pro.git](https://github.com/your-username/ddos-ninja-pro.git)
   cd ddos-ninja-pro
   

2. Установите зависимости:
   
   npm install
   

3. Настройте Telegram бота:
   Отредактируйте файл server.js, вставив ваш токен от BotFather:
   
   const bot = new Telegraf("ВАШ_ТОКЕН");
   

4. Запустите сервер:
   
   npm start
   
   После этого игра будет доступна по адресу: http://localhost:3000

## 📂 Структура папок
* /public — статические файлы (картинки маскотов, звуки, стили).
* server.js — серверная логика и API.
* game.js — игровая логика и Vue-компоненты.
* database.sqlite — локальная база данных.
